# mnemo v4.1 — C2 Provider Contract (registry) Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans. RIGID: superpowers:test-driven-development (red→green every task), superpowers:verification-before-completion. This is a **behaviour-preserving refactor** — the existing provider test suite is the safety net and must stay green *unchanged* at every step. On any test regression: superpowers:systematic-debugging (root-cause; do not weaken the existing assertions).

**Goal:** Replace the hand-edited `get_provider()` if/elif + 4 scattered capability tables (`DEFAULT_MODELS`, `keys.ENV_VAR`, `Config.providers`, `compaction.NATIVE_COMPACTION`) with one `ProviderDescriptor` dataclass + a `PROVIDERS` registry + `register_provider()`, so adding a provider = **one `register_provider(...)` call + one `stream()` impl class** instead of editing 5 files.

**Architecture:** Mirror `agent_tools.TOOLS` / `_register()` *exactly* (a module-level dict + a validating registrar that raises on dupes). Each of the 4 concrete provider modules self-registers a descriptor at import time; `providers/__init__.py` imports them at the bottom (define registry/registrar/descriptor/`BaseProvider` first, import concrete modules last — the standard register-at-bottom pattern, avoids circular import). `get_provider`, `keys.resolve_*`, `config` defaults, `DEFAULT_MODELS`, and `compaction.supports_native_compaction` all *derive from the registry*. New `GET /v1/providers` exposes it for C4 (v4.2). **No model defaults change** — the registry centralizes existing values, it does not re-decide them (YAGNI).

**Tech Stack:** Python dataclass + module dict (no new deps). pytest. FastAPI APIRouter for the new endpoint.

```python
from dataclasses import dataclass


@dataclass(frozen=True)
class ProviderDescriptor:
    name: str
    display_name: str
    impl_class: type
    env_var: str | None
    requires_key: bool
    default_model: str
    known_models: tuple[str, ...]
    base_url: str | None
    native_compaction_models: frozenset[str]


PROVIDERS: dict[str, ProviderDescriptor] = {}


def register_provider(desc: ProviderDescriptor) -> ProviderDescriptor:
    if desc.name in PROVIDERS:
        raise ValueError(f"duplicate provider registration: {desc.name}")
    PROVIDERS[desc.name] = desc
    return desc
```

```python
def get_provider(
    name: str, *, api_key: str | None = None, base_url: str | None = None
) -> BaseProvider:
    desc = PROVIDERS.get(name)
    if desc is None:
        raise ValueError(f"unknown provider: {name}")

    return desc.impl_class(**kwargs)
```
