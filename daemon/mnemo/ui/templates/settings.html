Retrieval settings

Tune how mnemo ranks memory entries against incoming queries. All edits save to ~/.claude/mnemo/settings.json.

Score formula: score = alpha·vector + beta·graph + gamma·recency + delta·type + epsilon·project + zeta·lexical

Weights don't have to sum to 1.0 — only the ratios between them affect ranking. unsaved changes

Scoring weights

six terms

Defaults

budget + decay

What each term means

alpha
cosine similarity to the query embedding
beta
boost for nodes adjacent to other top hits
gamma
recency decay (newer nodes rank higher)
delta
type priority based on inferred intent
epsilon
boost when the node is from the active project
zeta
literal token overlap on name + description
