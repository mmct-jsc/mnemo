Scoring weights

six terms

Defaults

budget + decay

What each term means

alpha
cosine similarity to the query embedding
beta
boost for nodes adjacent to other top hits
gamma
recency decay (newer nodes rank higher)
delta
type priority based on inferred intent
epsilon
boost when the node is from the active project
zeta
literal token overlap on name + description
